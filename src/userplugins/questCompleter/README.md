### üéØ QuestCompleter
Easily complete Discord Quests with a few clicks!
This uses an external script to complete quest which you can find [here](https://gist.github.com/aamiaa/204cd9d42013ded9faf646fae7f89fbb) full credit to Aamiaa for the script.

**Features:**
- Fetches available Discord quests automatically
- Supports multiple quest types (Watch Video, Play Game, Stream Game, Play Activity)
- Shows quest progress and time remaining
- One click quest completion using external script (Script by Aamiaa)
- Clean modal interface for quest management
- Real time quest status updates

**Supported Quest Types:**
- Watch Video
- Play Game (Desktop)
- Stream Game (Desktop) 
- Play Activity
- Watch Video (Mobile)
